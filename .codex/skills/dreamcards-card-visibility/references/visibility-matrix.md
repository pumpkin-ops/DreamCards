# Card Visibility Matrix

| Surface | Image | Player identity | Creator identity | Creator sequence | Tags | Counts | AI reasons | Inspiration notes |
|---|---:|---:|---:|---:|---:|---:|---:|---:|
| Hand / clue phase | yes | own seat only | no | no | no | no | no | no |
| Anonymous vote | yes | no | no | no | no | no | no | no |
| Round result | yes | submitter/voters | no | no | no | no | collapsed only | no |
| Replay | yes | yes | no by default | no by default | no | no | optional/collapsed | only if shared |
| Collection | yes | no | yes | yes | no | usage/collection allowed | no | no |
| Codex | yes | no | yes | yes | no | optional | no | no |
| Card detail/history | yes | no | yes | yes | no | yes | no | no |
| AI server input | optional URL | no | allowed if needed | allowed if needed | yes | no | n/a | no |

Gameplay text must never reveal a card title, tag, creator, public ID, sequence, collection count, or usage count.
